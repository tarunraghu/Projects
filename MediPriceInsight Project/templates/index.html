About Request Data Dump

This feature allows you to export or download the entire dataset of hospital charges and related information. Use this tool to obtain a comprehensive data dump for backup, analysis, or migration purposes. The exported data can be used for further processing, reporting, or integration with other systems.

Price Transparency Report
About Price Transparency Report

This report provides detailed pricing information for medical procedures across different healthcare providers, payers, and plans. The data helps patients and healthcare consumers make informed decisions about their healthcare costs by comparing prices for the same procedures across different facilities.
